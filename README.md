# Task Management Application

A professional task management application built with React, TypeScript, Redux Toolkit, and Tailwind CSS. Features include user authentication, task CRUD operations, dark mode, and responsive design.

## Features

- 🔐 User Authentication (Mocked with JWT)
- ✅ Task Management (Create, Read, Update, Delete)
- 🎨 Dark Mode Toggle
- 📱 Fully Responsive Design
- 🔍 Task Filtering and Search
- 💾 Persistent State (localStorage)
- 🧪 100% Test Coverage
- 🚀 MSW for API Mocking

## Tech Stack

- React 18
- TypeScript
- Redux Toolkit
- Tailwind CSS
- Formik & Yup
- MSW (Mock Service Worker)
- Jest & React Testing Library
- Vite

## Prerequisites

- Node.js (v18 or higher)
- npm or yarn
